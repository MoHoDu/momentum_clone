# url 

정적인 페이지를 1억개 가지고 있다면?
똑같은 파일인데 viewtopic.php.../이후의 값이 다른 페이지를 출력해줌

예시 주소)
http://opentutorials.org:3000/main?id=HTML&page-12

http - protocol 통신규칙
    프로토콜 (통신규칙) : 사용자가 서버에 접속할 때에 어떤 방식으로 
        통신할것인가
    http : 웹브라우저랑 웹서버가 서로 데이터를 주고받기 위해 만든 
        통신규칙
opentutorials.org - host(domain) 도메인 이름
    호스트 : 인터넷에 접속되어 있는 각각의 컴퓨터
    특정한 인터넷에 연결되어 있는 컴퓨터를 가리키는 주소
3000 - port 
    3000번 포트에 연결되어 있는 서버에 연결
    80번(웹서버) : 전세계적으로 80으로 약속 - 생략해도 이걸로 됨
main - path
    path : 컴퓨터 안에 있는 어떤 디렉토리 어떤 파일인지
id=HTML&page=12 - query string
    웹서버에게 어떤 데이터를 전달할 수 있음
    시작은 항상 ?로 시작
    값과 값은 &
    값의 이름과 값음 =로 구분하도록 약속
