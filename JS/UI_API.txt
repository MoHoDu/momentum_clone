API : Application Progrmming Interface
- 일반적인 사용자들이 사용하는 인터페이스는 아님
- 코드로 되어있는 인터페이스
- alert() 등...

javascript:alert("hello world");
javascript: 프로그래밍 언어의 명령어 이다.
alert() 경고창을 띄우는 명령어

# 자바스크립트 기본 API 레퍼런스 
ECMA Script === javascript
LiveScript가 원래 이름 --> JavaScript
--> 언어의 표준을 국제기구에게 위임 --> 위임된 국제기구가 ECMA
--> 상표권 문제로 JavaScript 이름을 ECMAScript로 표기
: JS 표준안을 발표
